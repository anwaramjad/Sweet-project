# دليل استخدام نظام إدارة محل الحلويات

## 🌟 مرحباً بك في نظام إدارة محل الحلويات

نظام شامل لإدارة محل الحلويات مع قاعدة بيانات متطورة تتضمن:
- تخزين أسعار المنتجات والمخزون
- إدارة العملاء والاشتراكات
- تسجيل الطلبات والمبيعات  
- تحليلات مفصلة للمبيعات والعملاء

## 🚀 تشغيل النظام

### الطريقة الأولى: تشغيل التطبيق الرسومي
```bash
python3 sweet_shop_sqlite.py
```

### الطريقة الثانية: تشغيل بدون واجهة رسومية (للاختبار)
```bash
python3 -c "from sweet_shop_sqlite import SweetShopDatabase; db = SweetShopDatabase(); print('النظام جاهز للاستخدام')"
```

## 📋 المميزات الرئيسية

### 1. إدارة المنتجات
- **27 منتج متنوع** موزع على 5 فئات:
  - حلويات باردة (10 منتجات)
  - حلويات ساخنة (5 منتجات)  
  - مشروبات باردة (5 منتجات)
  - مشروبات ساخنة (7 منتجات)
- تتبع المخزون تلقائياً
- تسجيل حركات البيع والشراء

### 2. إدارة العملاء
- تسجيل بيانات العملاء الكاملة
- نظام نقاط الولاء (نقطة لكل ريال)
- تتبع تاريخ المشتريات

### 4. إدارة الطلبات

- حساب الخصومات تلقائياً للمشتركين
- تتبع حالة الطلب

### 5. التحليلات والتقارير
- أكثر المنتجات مبيعاً

## 🎯 سيناريوهات الاختبار المقترحة

### سيناريو 1: بيع لعميل مشترك
1. افتح التطبيق
2. أدخل رقم الهاتف: `0501234567`
3. اضغط "بحث" - سيظهر أحمد مع خصم 15%
4. اختر منتجات (مثل: تشيز كيك، قهوة موكا)
5. أضف الكمية المطلوبة
6. لاحظ حساب الخصم تلقائياً
7. أكد الطلب

### سيناريو 2: إضافة عميل جديد
1. من قائمة "العملاء" → "إضافة عميل جديد"
2. أدخل البيانات
3. احفظ العميل
4. اصنع له طلب

### سيناريو 3: إنشاء اشتراك جديد
1. ابحث عن عميل غير مشترك
2. من قائمة "الاشتراكات" → "إنشاء اشتراك"
3. اختر نوع الاشتراك
4. أكد الاشتراك

### سيناريو 4: عرض التقارير
1. من قائمة "التقارير" اختر أي تقرير
2. راجع البيانات والإحصائيات
3. لاحظ تحليل الأداء

## 🛠️ استكشاف الأخطاء

### مشكلة: "لم يتم العثور على العميل"
**الحل:** تأكد من إدخال رقم الهاتف صحيح أو أضف العميل أولاً

### مشكلة: "الكمية أكبر من المخزون"
**الحل:** راجع المخزون المتاح أو قم بتقليل الكمية

### مشكلة: خطأ في قاعدة البيانات
**الحل:** امسح ملف `sweet_shop.db` وأعد تشغيل التطبيق

### مشكلة: التطبيق لا يفتح
**الحل:** تأكد من تثبيت `python3-tk`:
```bash
sudo apt install python3-tk
```
