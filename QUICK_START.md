```bash

```

```bash
python sweet_shop_management.py
```

## الأخطاء الشائعة وحلولها

### خطأ الاتصال بقاعدة البيانات
```
Error: Access denied for user 'root'@'localhost'
```
**الحل:** تأكد من أن MySQL يعمل وكلمة المرور صحيحة

### خطأ عدم وجود قاعدة البيانات
```
Error: Unknown database 'sweet_shop_db'
```
**الحل:** تشغيل ملف `database_schema.sql` أولاً

### خطأ في المكتبات
```
ModuleNotFoundError: No module named 'mysql'
```
**الحل:** تثبيت المكتبات المطلوبة باستخدام pip
