🍰 قائمة ملفات نظام إدارة محل الحلويات - النسخة المتطورة
================================================================

📁 الملفات الأساسية المطلوبة:
================================

🐍 sweet_shop_advanced.py
   - التطبيق الرئيسي المحدث
   - يحتوي على نظام تسجيل الدخول وطباعة الفواتير
   - حجم تقريبي: ~50 كيلوبايت

📋 requirements_advanced.txt  
   - قائمة المكتبات المطلوبة للنسخة المتطورة
   - pandas, matplotlib, reportlab, Pillow
   - حجم تقريبي: 200 بايت

🪟 run_advanced_sweet_shop.bat
   - سكريپت تشغيل للويندوز (محدث)
   - تثبيت تلقائي للمكتبات والتشغيل
   - حجم تقريبي: 3 كيلوبايت

🐧 run_advanced_sweet_shop.sh
   - سكريپت تشغيل للينكس/ماك (محدث)
   - تثبيت تلقائي للمكتبات والتشغيل
   - حجم تقريبي: 3 كيلوبايت

📖 ADVANCED_USER_GUIDE.md
   - دليل المستخدم الشامل للنسخة المتطورة
   - يشرح جميع المميزات الجديدة
   - حجم تقريبي: 15 كيلوبايت

📘 README_ADVANCED.md
   - ملف README مفصل للنسخة المتطورة
   - نظرة عامة وتعليمات التثبيت
   - حجم تقريبي: 20 كيلوبايت

🔧 تعليمات التحميل والتشغيل:
=============================

1️⃣ حمل الملفات الأساسية:
   - sweet_shop_advanced.py
   - requirements_advanced.txt
   - سكريپت التشغيل المناسب لنظامك

2️⃣ ضعها في مجلد واحد:
   mkdir sweet_shop_advanced
   cd sweet_shop_advanced

3️⃣ شغل النظام:
   - للويندوز: انقر نقراً مزدوجاً على run_advanced_sweet_shop.bat
   - للينكس/ماك: ./run_advanced_sweet_shop.sh

4️⃣ استخدم بيانات تسجيل الدخول:
   اسم المستخدم: admin
   كلمة المرور: admin123

📞 في حالة المشاكل:
==================

❌ خطأ في المكتبات:
pip install pandas matplotlib reportlab Pillow

❌ مشكلة في الواجهة:
sudo apt install python3-tk  # للينكس

❌ مشكلة في الصلاحيات:
chmod +x *.sh  # جعل السكريپت قابل للتنفيذ
