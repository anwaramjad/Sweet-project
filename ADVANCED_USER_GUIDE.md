# 🍰 دليل المستخدم - نظام إدارة محل الحلويات المتطور

## مرحباً بك في النسخة المتطورة! 🎉

تم تطوير نظام إدارة محل الحلويات ليصبح أكثر احترافية وسهولة في الاستخدام مع مميزات جديدة مذهلة.

---

## 🌟 المميزات الجديدة

### 🔐 نظام تسجيل الدخول الآمن
- **صفحة تسجيل دخول جميلة** مع تشفير كلمات المرور
- **إنشاء حسابات جديدة** مع زر Sign Up 
- **أدوار مختلفة** (مدير، كاشير)
- **حماية البيانات** مع تشفير SHA-256

### 🎨 تصميم متطور وجميل
- **ألوان احترافية** متناسقة
- **أيقونات تعبيرية** (إيموجي) لسهولة التنقل
- **تخطيط محسن** مع استخدام أفضل للمساحة
- **شريط حالة** يعرض معلومات المستخدم والوقت

### 🖨️ نظام طباعة الفواتير المتقدم
- **فواتير PDF احترافية** مع شعار المحل
- **تصميم جميل** مع ترقيم وتنسيق متقن
- **معلومات شاملة** (العميل، المنتجات، الخصومات)
- **حفظ تلقائي** مع اقتراح اسم الملف

### 💫 تجربة مستخدم محسنة
- **نوافذ منبثقة جميلة** للعمليات المختلفة
- **رسائل تأكيد واضحة** مع خيارات متعددة
- **تنبيهات ملونة** حسب نوع الرسالة
- **اختصارات لوحة المفاتيح** (Enter للبحث والتأكيد)

## 🚀 كيفية التشغيل

### للويندوز 🪟
1. **انقر نقراً مزدوجاً** على ملف `run_advanced_sweet_shop.bat`
2. أو افتح **Command Prompt** وشغل:
   ```cmd
   run_advanced_sweet_shop.bat
   ```

### للينكس/ماك 🐧🍎
1. افتح **Terminal** وشغل:
   ```bash
   ./run_advanced_sweet_shop.sh
   ```
2. أو:
   ```bash
   python3 sweet_shop_advanced.py
   ```

## 📱 كيفية استخدام النظام

### 1️⃣ تسجيل الدخول
1. **أدخل بياناتك** في صفحة تسجيل الدخول
2. أو **أنشئ حساب جديد** بالنقر على "Sign Up"
3. **اختر الدور المناسب** (مدير أو كاشير)

### 2️⃣ البحث عن العملاء
1. **أدخل رقم الهاتف** في حقل البحث
2. **اضغط Enter** أو انقر "🔍 بحث"
3. **أضف عميل جديد** إذا لم يكن موجوداً

### 3️⃣ إضافة المنتجات للسلة
1. **اختر منتج** من الجدول
2. **حدد الكمية** المطلوبة
3. **انقر "➕ إضافة للسلة"**

### 4️⃣ تأكيد الطلب وطباعة الفاتورة
1. **راجع السلة** والمجموع النهائي
2. **انقر "✅ تأكيد الطلب"**
3. **اختر طباعة الفاتورة** من النافذة المنبثقة
4. **احفظ الفاتورة** في المكان المناسب

## 🛠️ استكشاف الأخطاء

### مشاكل شائعة وحلولها:

#### ❌ خطأ في تسجيل الدخول
**الحل**: تأكد من صحة اسم المستخدم وكلمة المرور

#### ❌ لا تعمل الطباعة
**الحل**: تأكد من تثبيت مكتبة ReportLab:
```bash
pip install reportlab
```

#### ❌ مشكلة في التصميم
**الحل**: تأكد من تثبيت Tkinter:
```bash
# لنظام أوبونتو/ديبيان
sudo apt install python3-tk
```

#### ❌ قاعدة البيانات لا تعمل
**الحل**: تأكد من أن الملف `sweet_shop_advanced.db` قابل للكتابة

## 📋 متطلبات النظام
