## 🔧 خطوات التثبيت والتشغيل

### الخطوة 1: تحميل الملفات
1. قم بتحميل جميع الملفات المذكورة أعلاه
2. ضعها في مجلد واحد باسم `sweet_shop_project`

### الخطوة 2: تثبيت Python
تأكد من وجود Python 3.7 أو أحدث:
```bash
python3 --version
```

### الخطوة 3: تثبيت المكتبات المطلوبة

#### على Ubuntu/Debian:
```bash
sudo apt update
sudo apt install python3-tk
pip3 install pandas matplotlib
```

#### على Windows:
```bash
pip install pandas matplotlib tkinter
```

#### على macOS:
```bash
pip3 install pandas matplotlib
# tkinter عادة متوفر مع Python
```

### الخطوة 4: تشغيل التطبيق

#### تشغيل التطبيق الرئيسي (الواجهة الرسومية):
```bash
cd sweet_shop_project
python3 sweet_shop_sqlite.py
```

#### تشغيل العرض التوضيحي (واجهة نصية):
```bash
python3 demo.py
```

## 🔍 استكشاف الأخطاء

### مشكلة: "لا يمكن العثور على tkinter"
**الحل:**
```bash
# Ubuntu/Debian
sudo apt install python3-tk

# CentOS/RHEL
sudo yum install tkinter

# macOS
# tkinter متوفر عادة، جرب:
python3 -m tkinter
```

### مشكلة: "لا يمكن العثور على pandas أو matplotlib"
**الحل:**
```bash
pip3 install pandas matplotlib
```

### مشكلة: خطأ في قاعدة البيانات
**الحل:**
```bash
# احذف قاعدة البيانات الحالية وأعد إنشاؤها
rm sweet_shop.db
python3 add_test_customers.py
```

### مشكلة: لا تظهر النصوص العربية بشكل صحيح
**الحل:**
- تأكد من أن نظامك يدعم الخطوط العربية
- في Linux: `sudo apt install fonts-liberation fonts-dejavu`
