# نظام إدارة محل الحلويات

نظام شامل لإدارة محل الحلويات يشمل إدارة المنتجات والعملاء والاشتراكات مع تحليلات مفصلة للمبيعات.

## المميزات

### 📦 إدارة المنتجات
- تخزين أسعار المنتجات وتكلفتها
- تتبع المخزون تلقائياً
- تصنيف المنتجات (حلويات باردة، ساخنة، مشروبات)
- تسجيل حركات المخزون

### 👥 إدارة العملاء
- تسجيل بيانات العملاء الكاملة
- نظام نقاط الولاء
- تتبع تاريخ الشراء
- إحصائيات العملاء المفصلة

### 💳 نظام الاشتراكات
- **اشتراك أسبوعي**: خصم 10% لمدة أسبوع
- **اشتراك شهري**: خصم 15% مع توصيل مجاني
- **اشتراك ربع سنوي**: خصم 20% مع مزايا إضافية
- **اشتراك سنوي**: خصم 25% مع جميع المزايا

### 📊 التحليلات والتقارير
- **أكثر المنتجات مبيعاً**: تحديد المنتجات الأكثر شعبية
- **أقل المنتجات مبيعاً**: تحديد المنتجات التي تحتاج تسويق
- **تحليل العملاء**: معرفة أفضل العملاء وسلوكهم الشرائي
- **المبيعات الشهرية**: تتبع الأداء عبر الزمن
- **تحليل الفئات**: أداء كل فئة من المنتجات

## متطلبات النظام

### قاعدة البيانات
- MySQL Server 5.7+
- إنشاء قاعدة بيانات باسم `sweet_shop_db`

### المكتبات المطلوبة
```bash
pip install -r requirements.txt
```

## التثبيت والإعداد

### 1. إعداد قاعدة البيانات
```bash
# تشغيل MySQL وإنشاء قاعدة البيانات
mysql -u root -p < database_schema.sql
```

### 2. تثبيت المكتبات
```bash
pip install mysql-connector-python pandas matplotlib
```

### 3. تشغيل النظام
```bash
python sweet_shop_management.py
```

## بنية قاعدة البيانات

### الجداول الرئيسية
- **categories**: فئات المنتجات
- **products**: المنتجات وأسعارها
- **customers**: بيانات العملاء
- **subscription_types**: أنواع الاشتراكات
- **customer_subscriptions**: اشتراكات العملاء
- **orders**: الطلبات
- **order_items**: تفاصيل الطلبات
- **payments**: المدفوعات
- **product_reviews**: تقييمات المنتجات
- **inventory_transactions**: حركات المخزون

### المشاهد التحليلية
- **top_selling_products**: أكثر المنتجات مبيعاً
- **least_selling_products**: أقل المنتجات مبيعاً
- **customer_analytics**: تحليل العملاء
- **monthly_sales_analysis**: تحليل المبيعات الشهرية
- **category_analysis**: تحليل الفئات

## كيفية الاستخدام

### إضافة عميل جديد
1. انتقل إلى قائمة "العملاء" → "إضافة عميل جديد"
2. أدخل البيانات المطلوبة (الاسم والهاتف إجباريان)
3. اضغط "حفظ"

### إجراء عملية بيع
1. ابحث عن العميل بالهاتف
2. اختر المنتجات من القائمة
3. حدد الكمية واضغط "إضافة للسلة"
4. اضغط "تأكيد الطلب" لإكمال البيع

### إنشاء اشتراك
1. اختر العميل أولاً
2. انتقل إلى "الاشتراكات" → "إنشاء اشتراك"
3. اختر نوع الاشتراك المناسب
4. اضغط "إنشاء الاشتراك"

### عرض التقارير
- انتقل إلى قائمة "التقارير"
- اختر نوع التقرير المطلوب
- سيتم عرض البيانات في نافذة منفصلة
